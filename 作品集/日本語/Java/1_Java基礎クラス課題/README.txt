* 本Java基礎クラス課題は、とある日本の企業様が開催した【IT保證班体験レッスン】に参加させていただいた際に作成および提出した課題のまとめとなる
* 【参加時期】2024年10月20日～26日

* 【体験レッスンの参加形式】
  * 履歴書および志望動機などを提出し、一回目の面接を受ける
  * 専用サイトにて動画または文字から学習する
  * 講義内容に沿ってJavaコードを操作し実演する
  * 各章の終わりに1~3問の課題が下され、どのように解くか学習してきた内容を利用して自力で導き出す
  * 実演内容および課題ごと先方様に渡し、学習状況についてフィードバックをいただく
  * コーディングにわからない箇所がある場合、コメント形式または専属の先生方と時間を予約して相談する
  * 体験一週間後、最終面接を受け合否判定を仰ぐ
  * 合格の場合、該企業の元で引き続きコーディングについてを勉強し、6ヶ月以内に日本のエンジニア職のお仕事を斡旋していただける

* 【体験レッスン内容】
  1. 基礎Javaの概念・操作
  2. Java Web・フレームワークおよびHTML/CSSの基本概念・操作体験
  3. データベースの基本概念（SQLなどへの操作学習は体験レッスン対象外）

* 【体験レッスン参加結果】
  * 「レッスン内容に対する理解度が高くコードも綺麗にまとめられている」とのフィードバックをいただきました。
  * 体験後の面接にて、やはり1年以上のブランクがあることに懸念を抱いたため就職保障ができかねるとお断りされました。
  * 私は日本で就職して生活していきたい夢を叶えるため、これを機に、本格的に本『BennkyouNote』を充実させ、将来の転職活動におけるポートフォリオとして、またはコーディングの際の手引きとして活用していきたいと考えました。
